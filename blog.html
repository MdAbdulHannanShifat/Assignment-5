Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web pages.
It provides a representation of the page that allows programs to modify the document's structure, appearance, and content.
The DOM structures the document as nodes and objects, enabling programming languages to interact with the page.

How do you select an element from the DOM?

1. By using getElementById
2. By using getElementsByClassName
3. By using getElementsByTagName
4. By using querySelector
5. By using querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is the technique of utilizing event propagation (bubbling) to manage events at a parent or higher level in the DOM, instead of directly on the element where the event occurs. This approach lets us add a single event listener for elements that are present or will be dynamically added in the future.
